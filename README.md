## Development Setup

### Prerequisites

- Install [Node.js] which includes [Node Package Manager][npm]
- Install Composer

### Setting Up a Project
#### Frontend
Install the dependency globally, you can use yarn or npm:

Using NPM :
```
npm install 
```

Run the application:

```
cd [PROJECT NAME]
npm run dev
```

Using YARN :
```
yarn install 
```

Run the application:

```
cd [PROJECT NAME]
yarn dev
```
#### Backend
Install the dependency globally
```
composer install 
```
